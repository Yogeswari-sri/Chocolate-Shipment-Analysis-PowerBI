# Chocolate Shipment Analysis Dashboard

This repository showcases a **Power BI dashboard** designed to analyze chocolate sales, revenue, and profit across regions, categories, and salespersons.  
The project demonstrates how business intelligence tools can transform raw shipment data into actionable insights.

## 📂 Contents
- `Chocolate_Shipment.pbix` – Power BI dashboard file
- `Data Sources` – Sales, Regional, and Salesperson datasets
- `Project_Report.md` – Detailed documentation of objectives, visuals, and findings
- Screenshots of dashboard components

## 🎯 Objectives
- Track **total revenue, profit, and boxes shipped**
- Compare **regional performance** (APAC, Americas, Europe)
- Identify **top-performing salespersons**
- Analyze **yearly profit trends** (2023 vs 2024)
- Provide filters for **sales phase** and **order status**

## 📊 Dashboard Components
- **KPIs**
  - Total Revenue: 44.69M
  - Total Profit: 20.24M
  - Total Boxes Shipped: 4M
- **Visuals**
  - Profit by Region (Pie Chart)
  - Total Profit by Country (Bar Chart)
  - Profit by Salesperson (Treemap)
  - Revenue by Category (Bar Chart)
  - Total Revenue by Salesperson (Horizontal Bar)
  - Profit Trend by Year (Line Chart)
- **Filters**
  - Sales Phase: Start, Mid, Fest, End
  - Order Status: Placed, Shipped, Delivered, Cancelled

## 🔍 Insights
- Europe contributes the highest profit share (~53%)
- Salespersons **Gunar Cockshott** and **Dennison Cross** lead in revenue and profit
- Profits declined from 2023 (~8M) to 2024 (~7M), signaling a need for strategy review

## 🚀 Conclusion
The Chocolate Shipment Analysis dashboard enables:
- Quick identification of **regional strengths and weaknesses**
- Recognition of **top performers**
- Monitoring of **yearly profit trends**

This project demonstrates how **Power BI** can be leveraged for **data-driven decision-making** in sales and marketing.
